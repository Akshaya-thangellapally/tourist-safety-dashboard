# Tourist Safety System

## About the Project

The Tourist Safety System is a web-based application developed to help travelers access emergency services quickly and efficiently. Tourists often face challenges in locating nearby hospitals, police stations, or fire stations in unfamiliar places. This project addresses that problem by providing live location tracking, nearby emergency service search, route navigation, voice commands, multilingual support, and an SOS feature through a simple and user-friendly interface.

This project was developed as part of our B.Tech Academic Mini Project.

---

## Features

* Live location tracking using the browser's Geolocation API
* Search for nearby hospitals
* Search for nearby police stations
* Search for nearby fire stations
* Interactive map using Leaflet and OpenStreetMap
* Automatic route generation to the nearest emergency service
* Distance calculation and estimated travel time
* Voice command support
* Multilingual interface

  * English
  * Hindi
  * Telugu
  * Tamil
* SOS alert through WhatsApp
* Responsive user interface
* Online and offline system status monitoring

## How It Works

1. The application detects the user's current location using the browser's Geolocation API.
2. The live location is displayed on an interactive map.
3. Users can search for nearby hospitals, police stations, or fire stations.
4. The system calculates the nearest emergency services based on the user's location.
5. A driving route and estimated travel time are generated using the OSRM Routing API.
6. Users can also search using voice commands such as "Hospital", "Police", or "Fire".
7. In case of an emergency, the SOS button allows users to share their live location through WhatsApp.
